Documentation

phc's documentation has been split into more appropriate catagories.

User Manual

The User Manual details how to configure, compile, install, and test phc. It also discusses how to use PHC's XML output, and phc's command line options.

Developer Manual

The Developer Manual describes how to create plugins to analyse or manipulate PHP using phc, and how to modify phc's internals.
