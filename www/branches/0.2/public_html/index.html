What is phc?

phc is an open source compiler for PHP with support for plugins. In addition, it can be used to pretty-print or obfuscate PHP code, as a framework for developing applications that process PHP scripts, or to convert PHP into XML and back, enabling processing of PHP scripts using XML tools.

Authors

The main authors of phc are Edsko de Vries, Paul Biggar and John Gilbert.

The following alphabetical list of people have contributed to phc: Andras Biczo, Andreas Korthaus, Conor McDermottroe (BSD support), Daniel Barreiro, Daniel Fabian, Dan Libby, David Abrahamson, Eric Bouwers, Jakub Suchy, Lorenzo Bettini (gengetopt), Marc Eisenbarth, Matthias Kleine (unparser), Sven Klemm, Tim Van Holder.
