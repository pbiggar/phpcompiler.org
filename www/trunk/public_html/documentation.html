Documentation

User Manual

The User Manual details how to configure, compile, install, and test phc. It also discusses how to use phc's XML output, and phc's command line options. The Developer Manual describes how to create plugins to analyse or manipulate PHP using phc, and how to modify phc's internals. Both the user and the developer manual are available in the same document. The latest version (0.2.0) is available as a PDF or online in HTML. We don't recommend any other version, though previous versions are archived.

Other documentation:
